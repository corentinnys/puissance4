RULES

OBJECTIVE

Be the first player to connect 4 of the same colored discs in a row (either vertially, horizontally or diagonally)

HOW TO PLAY

  1. Red goes first in the first game
  2. Players must alternate turns, and only disc can be dropped in each turn
  3. The game end when there is a 4-in-a-row or stalemate
  4. The starter of the previous game goes second on the next game
